ABCL Sentinel AI – Multi-Agent Collections Fraud Detection Platform

Technologies: Python, FastAPI, LangGraph, LangChain, OpenAI GPT-4, PostgreSQL, Neo4j, SQLAlchemy, React, Docker, JWT, Redis

Description

Designed and developed an enterprise-grade AI-powered fraud detection platform using LangGraph Multi-Agent Architecture to monitor loan repayment fraud, agent collusion, third-party funding, and suspicious financial transactions.
Developed an 8-Agent AI workflow where specialized AI agents collaboratively performed repayment anomaly detection, source account verification, risk scoring, fraud investigation, alert generation, and case management.
Built secure REST APIs using FastAPI with JWT authentication and Role-Based Access Control (RBAC) supporting Admin, Collection Manager, Investigator, Auditor, and Agent portals.
Designed and implemented a 40+ table PostgreSQL database to manage customers, loans, repayments, fraud alerts, investigations, audit logs, AI predictions, collection targets, employee monitoring, and risk analytics.
Integrated Neo4j Graph Database to detect hidden relationships between customers, funding accounts, collection agents, agencies, and suspicious transaction networks for fraud investigation.
Developed AI-powered fraud detection modules for:
Month-end repayment spike detection
Common funding account detection
Third-party repayment monitoring
Agent and employee collusion detection
Source account mismatch analysis
Customer risk scoring
Automated fraud case generation
Built intelligent investigation workflows with automated case creation, evidence management, timeline tracking, approval workflow, and audit history.
Developed scalable data ingestion pipelines for processing high-volume repayment records and customer data using SQLAlchemy and PostgreSQL bulk loading.
Containerized the complete application using Docker, Nginx, and Docker Compose for production-ready deployment.
Implemented live dashboards providing fraud alerts, KPIs, risk analytics, branch performance, and agent-wise fraud monitoring.
